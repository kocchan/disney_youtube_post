Privacy Policy

Last updated: 2026-07-05

This application ("the App") is a personal automation tool operated by a single individual to publish short-form video content to the operator's own TikTok account via the TikTok Content Posting API. The App is not distributed to or used by other users.

Data Collected

The App stores, locally on the operator's own machine, OAuth credentials (access token and refresh token) issued by TikTok for the sole purpose of authenticating API requests to publish the operator's own videos. No data about other individuals is collected, stored, or shared.

Data Use

Credentials are used exclusively to call the TikTok Content Posting API in order to upload videos to the operator's own TikTok account. No data is sold, shared with third parties, or used for advertising.

Data Retention & Deletion

Credentials are stored locally and can be deleted at any time by the operator. Revoking the App's access from the TikTok account settings immediately invalidates the stored tokens.
